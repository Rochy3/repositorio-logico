# Repositorio Educativo de Pensamiento Lógico

## Descripción del proyecto

Este proyecto es un prototipo funcional básico de un repositorio educativo digital orientado al fortalecimiento del pensamiento lógico, el razonamiento matemático y la demostración preformal en estudiantes de grados 6° a 9°.

El repositorio integra una biblioteca de recursos descargables, una zona de retos y juegos, módulos interactivos, preguntas de selección múltiple, retroalimentación automática, puntaje e insignias. Su propósito es acercar a los estudiantes a la lógica matemática y a la justificación de ideas de una manera visual, sencilla y participativa.

## Objetivo

Diseñar e implementar un prototipo web interactivo que permita a los estudiantes explorar materiales de apoyo, resolver retos lógicos y practicar actividades relacionadas con patrones, razonamiento intuitivo, afirmaciones lógicas y demostración preformal.

## Público objetivo

El prototipo está dirigido a estudiantes de educación básica, especialmente de grados 6° a 9°, que están iniciando el tránsito entre el razonamiento intuitivo y el razonamiento deductivo.

## Tecnologías utilizadas

- HTML5: estructura de la página web.
- CSS3: diseño visual, colores, tarjetas, botones y adaptación a pantallas pequeñas.
- JavaScript: interactividad, preguntas, retroalimentación automática, puntaje, insignias y juegos.
- PDF: materiales descargables de apoyo para los estudiantes.

## Funcionalidades principales

### 1. Página de inicio

Presenta el nombre del proyecto y una explicación breve sobre el uso del repositorio. Desde allí el estudiante puede explorar la biblioteca, participar en retos y entrar a los módulos interactivos.

### 2. Biblioteca de recursos

Incluye tres guías descargables en PDF:

- Guía 1: Introducción a la lógica.
- Guía 2: Patrones y razonamiento.
- Guía 3: Primeras demostraciones.

Estos recursos funcionan como material informativo antes de realizar las actividades interactivas.

### 3. Zona de retos y juegos

El prototipo incluye actividades cortas para motivar la participación del estudiante:

- Reto del detective lógico: actividad de razonamiento para elegir una conclusión correcta.
- Juego de ordenar la demostración: juego en el que el estudiante debe seleccionar pasos en el orden adecuado para construir una justificación matemática sencilla.

### 4. Módulos interactivos

El repositorio contiene tres módulos principales:

#### Módulo 1: Patrones y razonamiento intuitivo

Trabaja la observación de regularidades, secuencias y patrones numéricos. El estudiante debe identificar reglas y justificar sus respuestas.

#### Módulo 2: Lógica si... todos

Permite practicar relaciones lógicas mediante afirmaciones que usan estructuras como “si... entonces” y expresiones generales como “todos”.

#### Módulo 3: Demostración preformal

Introduce explicaciones sencillas para justificar ideas matemáticas, usando ejemplos, contraejemplos y razonamientos comprensibles para el nivel escolar.

### 5. Retroalimentación automática

Después de responder cada pregunta, el sistema muestra un mensaje que indica si la respuesta fue correcta o si debe revisarse. Esto permite que el estudiante identifique sus aciertos y errores de manera inmediata.

### 6. Puntaje e insignias

Al finalizar cada módulo, el sistema muestra el puntaje obtenido y asigna una insignia de acuerdo con el desempeño:

- 3 respuestas correctas: Explorador lógico.
- 2 respuestas correctas: Buen razonador.
- 0 o 1 respuestas correctas: Sigue practicando.

## Cómo ejecutar el proyecto

### Opción 1: Abrir directamente el archivo HTML

1. Descargar o copiar la carpeta del proyecto.
2. Abrir la carpeta `repositorio-logico`.
3. Hacer doble clic sobre el archivo `index.html`.
4. El prototipo se abrirá en el navegador.

### Opción 2: Usar Visual Studio Code con Live Server

1. Abrir Visual Studio Code.
2. Seleccionar `File > Open Folder`.
3. Escoger la carpeta `repositorio-logico`.
4. Abrir el archivo `index.html`.
5. Hacer clic en `Go Live`.
6. Probar el repositorio desde el navegador.

## Cómo usar el repositorio

1. Leer la introducción de la página principal.
2. Consultar o descargar las guías de la biblioteca de recursos.
3. Resolver los retos y juegos de la zona interactiva.
4. Ingresar a los módulos.
5. Responder las preguntas de cada módulo.
6. Revisar la retroalimentación automática.
7. Observar el puntaje y la insignia obtenida al finalizar.

## Enfoque pedagógico

El prototipo se basa en una transición gradual desde el razonamiento intuitivo hacia la deducción lógica. Por eso, las actividades no empiezan con demostraciones formales complejas, sino con ejemplos, patrones, contraejemplos, frases lógicas y justificaciones sencillas.

La estructura del repositorio busca que el estudiante no solo seleccione una respuesta, sino que también entienda la razón de esa respuesta mediante retroalimentación inmediata.

## Estado del proyecto

El proyecto se encuentra en fase de prototipo funcional básico. La versión actual permite navegar por recursos, jugar retos sencillos, responder módulos interactivos y recibir retroalimentación automática.
